Pictionary

A real-time pixel like word-guessing game for fun mainly focused on HTML Canvas. Used Action Cable which integrates WebSocket with Rails to stream Canvas data and chat messages across multiple players. Used Redis as adapter to run Action Cable in production.

Technologies used: React, Tailwind CSS, Redux, Ruby on Rails, MVC, Redis, Action Cable, PostgreSQL, Heroku

Blog Website

A basic blog website for personal use based on RESTful APIs. Used Bcrypt to achieve password encryption-decryption and Redux-Persist library to allow persistent login state.

Technologies used: React, Chakra UI, Redux, Redux-Persist, Ruby on Rails, SQLite

GitHub Repo: https://github.com/horus2121/blog_rails.git

Chat Command Line Interface (CLI)

A chat Command Line Interface (CLI) where people can chat in a public room with additional features. Utilized multi-threading to achieve better user experience and Rake to manage various tasks.

Technologies used: Ruby, Rake, Multi-threading

GitHub Repo: https://github.com/horus2121/chat_cli_ruby.git
